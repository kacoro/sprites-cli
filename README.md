# sprites-cli
A Sprites Generator.Composite images to Sprites And generate json or css

# How to install
```
npm install -g cc-sprites-cli


```

# script
```
# run

cc-sprites-cli 

# help
cc-sprites-cli  -h

# set target and output 
cc-sprites-cli  -s

# drop black backgournd
cc-sprites-cli  -d

# rename
cc-sprites-cli  -n sprites

# version
cc-sprites-cli  -V

```
